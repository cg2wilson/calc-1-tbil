    Section 4.1 I1: Antiderivatives

    Objectives

    I1
    I can compute the indefinite integral/antiderivative of a function using rules and appropriate notation.

    Pre-Class Preparation.

    Read the following sections of the text:
    • Section 4.10: Antiderivatives
      Read all sections

    Subsection 4.1.1 In-Class Activities

    Definition 4.1.1. Antiderivative/Indefinite Integral.

    The function \(F(x)\) is said to be an antiderivative of the function \(f(x)\) on the interval \(I\) if \(\dfrac{d}{dx}[F(x)] = f(x)\text{.}\) An antiderivative is often called an indefinite integral, and the notation is given by
    \begin{equation*} \ds \int f(x)\,dx = F(x) + C \end{equation*}
    where \(C\) is an arbitrary constant.

    Activity 4.1.1.

    Which of the following could be antiderivatives for the function \(f(x) = \cos (x)\text{?}\) In other words, which option or options could be a solution to
    \begin{equation*} \ds \int \cos (x)\, dx \end{equation*}
    1. \(\displaystyle \tan (x)\)
    2. \(\displaystyle -\sin (x)\)
    3. \(\displaystyle -\sin (x) + 3\)
    4. \(\displaystyle \sin (x) - 3\)
    5. \(\displaystyle \sin (x) + \pi\)
    6. \(\displaystyle \cos (x)\)
    7. \(\displaystyle \sin(x) - \pi\)
    8. \(\displaystyle \tan (x) +3\)

    Activity 4.1.2.

    This activity will develop the power rule for indefinite integrals.
    (a)
    Rewrite the power rule for derivatives: \(\dfrac{d}{dx}[x^n] = \cdots\)
    (b)
    Consider the function \(f(x) = 2x\text{.}\) Which function could be an antiderivative for \(f(x)\text{?}\)
    1. \(\displaystyle F(x) = 2\)
    2. \(\displaystyle F(x) = \dfrac{1}{2}x^2\)
    3. \(\displaystyle F(x) = \dfrac{1}{3}x^2\)
    4. \(\displaystyle x^2\)
    (c)
    Repeat the process for \(f(x) = x^2\text{.}\)
    1. \(\displaystyle x^3\)
    2. \(\displaystyle \dfrac{1}{3}x^3\)
    3. \(\displaystyle \dfrac{1}{2}x^3\)
    4. \(\displaystyle \dfrac{1}{3}x^2\)
    (d)
    Once more, repeat with \(f(x) = x^3\text{.}\)
    1. \(\displaystyle x^4\)
    2. \(\displaystyle \dfrac{1}{3}x^4\)
    3. \(\displaystyle \dfrac{1}{4}x^3\)
    4. \(\displaystyle \dfrac{1}{4}x^4\)
    (e)
    Based on parts (a)-(d), what kind of pattern do you believe is viable to describe the power rule for indefinite integrals? Does this work for all numbers, or is there a specific one for which the rule fails?

    Remark 4.1.2.

    We now note that whenever we know the derivative of a function, we have a function-derivative pair, so we also know the antiderivative of a function. For instance, we determined a derivative pair in Activity 4.1.1. Every elementary derivative rule leads us to a function-derivative pair, and thus to a known antiderivative/indefinite integral.

    Activity 4.1.3.

    In the following activity, we work to build a list of basic functions whose antiderivatives we may already know.
    Use your knowledge of derivatives of basic functions to complete the table of antiderivatives below, Table 4.1.3.

    Activity 4.1.4.

    The Sum/Difference and Constant Multiple rules for derivatives also apply to indefinite integrals:
    \begin{equation*} \ds \int c\cdot f(x)\, dx = c \cdot \int f(x)\, dx \end{equation*}
    and
    \begin{equation*} \ds \int [f(x)\pm g(x)]\, dx = \int f(x)\, dx \pm \int g(x)\, dx \end{equation*}
    Use this information to compute the indefinite integral for the functions below.
    (a)
    \(f(x) = 5\sin(x) - 4x^2\)
    (b)
    \(g(t)=-3\csc(t)\cot(t) + 3(2^x)\)
    (c)
    \(h(x)=e^x - x^e + 3 - \dfrac{1}{x}\)
    (d)
    \(j(k) = \dfrac{4}{1+k^2} + \dfrac{\pi}{\sqrt{4-4k^2}}\)
    Hint: think about factoring a value out of the root...
    (e)
    \(W(x) = \cos (x) + 7\sin(x) - \sec^2(x)\)

    Activity 4.1.5.

    When integrating, we must include the \(+C\) term because, without extra information, we can’t know the antiderivative for certain. However, we can do this once we know information about the antiderivative. These kinds of problems, called initial value problems, are a type of ordinary differential equation.
    (a)
    Consider the function \(f'(x) = x^2 + 2x\text{.}\) Which of the following equations for \(f(x)\) satisfies the differential equation?
    1. \(\displaystyle f(x) = \frac{x^3}{3} + x^2 + 4\)
    2. \(\displaystyle f(x) = 2x + 2\)
    3. \(\displaystyle f(x) = \frac{x^3}{3} + x^2 + 10\)
    4. \(\displaystyle f(x) = \frac{x^3}{3} + x^2\)
    5. \(\displaystyle f(x) = 2x\)
    (b)
    Write a function for \(f(x)\) which captures the most general antideriative, i.e. the indefinite integral.
    (c)
    Now assume that we know \(f(0) = 10\text{.}\) What is the appropriate expression for \(f(x)\) knowing this piece of information.
    (d)
    Which of the following functions satisfies the initial value problem
    \begin{equation*} \dfrac{dy}{dx} = \sin(x),\quad y(0) = 1 \end{equation*}
    1. \(\displaystyle y(x) = \cos(x)\)
    2. \(\displaystyle y(x) = \cos(x) + 2\)
    3. \(\displaystyle y(x) = \cos(x) + 1\)
    4. \(\displaystyle y(x) = -\cos(x)\)
    5. \(\displaystyle y(x) = -\cos(x) + 2\)
    6. \(\displaystyle y(x) = -\cos(x) + 1\)

    Activity 4.1.6.

    One of the applications of initial value problems is calculating the distance traveled from a point based on the velocity of the object, or computing velocity given acceleration.
    (a)
    Given that the velocity of the of an object in km/hr is approximated by \(v(t) = \cos(t) + 1\text{,}\) what is the approximate distance travelled by the object after 1 hour?
    1. \(s(1) \approx 1\) km
    2. \(s(1) \approx 0.1585\) km
    3. \(s(1) \approx 1.8415\) km
    4. \(s(1) \approx 2.3415\) km
    (b)
    Recall that to find the acceleration of an object, you can take the derivative of the velocity of an object. Let use say we have the acceleration of a falling object in m/s\(^2\) given by \(a(t) = -9.8\text{.}\) What is the velocity of the falling object, if the initial velocity is given by \(v(0) = 0\) m/s?
    1. \(\displaystyle v(t) = -9.8t\)
    2. \(v(t) = -9.8t\) m/s
    3. \(v(t) = 9.8t\) m/s
    4. \(v(t) = 9.8t+1\) m
    (c)
    Now, what is the position of the object from the previous part, if its initial position is given by \(s(0) = 10\) m?
    1. \(s(t) = 4.9t + 10\) m
    2. \(s(t) = -4.9t^2 + 14.9\) m
    3. \(s(t) = -4.9t^2 + 10\) m
    4. \(s(t) = 4.9t + 5.1\) m

    Activity 4.1.7.

    Knowing that acceleration due to gravity is given by \(a(t) = -9.8\) m/s\(^2\text{,}\) for an object with initial velocity \(v_0\) and initial position \(s_0\text{,}\) write an expression which gives the object’s position at time \(t\text{.}\)
