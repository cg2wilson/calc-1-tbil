Definition 1.1.1. Limit.

Suppose the function \(f(x)\) is defined when \(x\) is near the input \(a\text{.}\) We write
\begin{equation*} \lim_{x\to a} f(x) = L \end{equation*}
if we can make the values of \(f(x)\) arbitrarily close to \(L\) by restricting \(x\) to be sufficiently close to \(a\text{,}\) but not equal to \(a\) itself.
In this case, we say that the limit of \(f\) as \(x\) approaches \(a\) is \(L\text{.}\)
