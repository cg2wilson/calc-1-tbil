Activity 4.2.1.

Determine the exact value of the definite integrals below; some potentially helpful geometric formulas are:
  • Triangle: \(A = \dfrac{1}{2}b\cdot h\)
  • Circle: \(A = \pi r^2\)
  • Rectangle: \(A = b\cdot h\)
If necessary, sketch the graph of the function.
(a)
\(\ds \int_0^2 x+1\, dx\)
(b)
\(\ds \int_1^3 f(x)\, dx\text{,}\) where \(f(x) = \begin{cases} 2 & x\lt 2 \\ -2x + 6 & x\geq 2 \end{cases}\)
(c)
\(\ds \int_{-1}^1 \sqrt{1-x^2}\, dx\)
Hint: The function is given by \(y=\sqrt{1-x^2}\text{;}\) if you aren’t sure about graph, try rearranging so that there aren’t any roots... is that a familiar expression?
