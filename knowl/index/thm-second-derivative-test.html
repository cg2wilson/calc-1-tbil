Theorem 3.3.11. Second Derivative Test.

Suppose that \(x=c\) is a critical input for \(f(x)\) and that \(f''(x)\) is continuous at \(x=c\text{.}\) Then,
  1. If \(f''(c) \lt 0\text{,}\) then \(f(x)\) has a local maximum at \(x=c\text{.}\)
  2. If \(f''(c) \gt 0\text{,}\) then \(f(x)\) has a local minimum at \(x=c\text{.}\)
  3. If \(f''(c) = 0\text{,}\) the Second Derivative Test is inconclusive.
