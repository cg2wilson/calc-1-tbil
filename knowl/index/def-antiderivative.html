Definition 4.1.1. Antiderivative/Indefinite Integral.

The function \(F(x)\) is said to be an antiderivative of the function \(f(x)\) on the interval \(I\) if \(\dfrac{d}{dx}[F(x)] = f(x)\text{.}\) An antiderivative is often called an indefinite integral, and the notation is given by
\begin{equation*} \ds \int f(x)\,dx = F(x) + C \end{equation*}
where \(C\) is an arbitrary constant.
