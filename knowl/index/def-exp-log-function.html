Definition 0.2.11. Exponential & Logarithmic Function.

An exponential function is a function of the form
\begin{equation*} f(x) = b^x \end{equation*}
where the base \(b\) is a positive constant.
A logarithmic function is a function of the form
\begin{equation*} g(x) = \log_b(x) \end{equation*}
where the base \(b\) is a positive constant.
