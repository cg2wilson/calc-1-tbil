    Chapter 4 Integrals (I)

    Objectives

    NEED TO CHANGE/ADD THIS
    I1
    I can compute the indefinite integral/antiderivative of a function using rules and appropriate notation
    I2
    I can use Riemann sums to approximate the area under curves and use appropriate notation to do so
    I3
    I can compute definite integrals using the definition of the definite integral, geometry, or properties and contextually interpret my answer
    I4
    I can compute the derivative of an integral-valued function using the First Fundamental Theorem of Calculus
    I5
    I can use the Second Fundamental Theorem of Calculus to compute definite integrals
    I6
    I can use substitution to compute definite or indefinite integrals
    I7
    I can determine the area between two curves
