    Section 5.1 VE1: Working with Vectors

    Objectives

    VE1
    I can identify, create, combine, modify, and apply vectors in two- or three-dimensional space.

    Pre-Class Preparation.

    Read the following sections of the text:
    • Volume 3, Section 2.1: Vectors in the Plane
      Read all sections
    • Volume 3, Section 2.2: Vectors in Three Dimensions
      Read all sections

    Subsection 5.1.1 In-Class Activities

    Definition 5.1.1. Vector.

    A vector is a quantity which has both magnitude/length and direction. We will denote a vector using boldface type, as \(\textbf{v}\text{.}\) Vectors are generally depicted visually as an arrow beginning at an initial point (\(A\) below) and ending at a terminal point (\(B\) below).

    Activity 5.1.1.

    Vectors can be decomposed into components- horizontal and vertical. See the diagram below.
    (a)
    If the length of the horizontal leg is \(a\text{,}\) and the length of the vertical leg is \(b\text{,}\) which is the best description for the magnitude of \(\textbf{v}\text{,}\) denoted \(||\textbf{v}||\text{?}\)
    1. \(\displaystyle a + b\)
    2. \(\displaystyle \sqrt{b^2 - a^2}\)
    3. \(\displaystyle \sqrt{a^2 + b^2}\)
    4. \(\displaystyle a^2 + b^2\)
    (b)
    Which expression best identifies the direction of the vector, denoted \(\theta\) in the diagram?
    1. \(\displaystyle \inv{\tan}\lrpar{\dfrac{a}{b}}\)
    2. \(\displaystyle \inv{\tan}\lrpar{\dfrac{b}{a}}\)
    3. \(\displaystyle \inv{\sin}\lrpar{\dfrac{a}{b}}\)
    4. \(\displaystyle \inv{\sin}\lrpar{\dfrac{b}{a}}\)
    5. \(\displaystyle \inv{\cos}\lrpar{\dfrac{a}{b}}\)
    6. \(\displaystyle \inv{\cos}\lrpar{\dfrac{b}{a}}\)
    (c)
    Use trigonometry and/or part (a) to find expressions for the length of \(a\) and the length of \(b\) in terms of the direction and magnitude of \(\textbf{v}\text{.}\)

    Remark 5.1.4.

    A vector \(\textbf{v}\) with horizontal component \(x\) and vertical component \(y\) is written in component form as \(\lrangle{x,y}\text{.}\)

    Remark 5.1.5.

    Vectors can be added and subtracted like regular numbers. Addition happens component-wise; for example, adding the vectors \(\lrangle{1,3} + \lrangle{-4,2}\) gives the resultant vector \(\lrangle{1-4,3+2} = \lrangle{-3,5}\)

    Activity 5.1.4.

    Let \(\textbf{u} = \lrangle{-2,1}\) and \(\textbf{v} = \lrangle{0,3}\text{.}\) Find the following vectors.
    (a)
    \(\textbf{u} + \textbf{v}\)
    1. \(\displaystyle \lrangle{2,2}\)
    2. \(\displaystyle \lrangle{-2,4}\)
    3. \(\displaystyle \lrangle{1,1}\)
    4. \(\displaystyle \lrangle{-1,3}\)
    (b)
    \(\textbf{u} - \textbf{v}\)
    1. \(\displaystyle \lrangle{-2,-2}\)
    2. \(\displaystyle \lrangle{2,2}\)
    3. \(\displaystyle \lrangle{-5,1}\)
    4. \(\displaystyle \lrangle{5,-1}\)
    (c)
    \(2\textbf{u} - \dfrac{1}{2}\textbf{v}\)
    1. \(\displaystyle \lrangle{2,5}\)
    2. \(\displaystyle \lrangle{-4,\dfrac{5}{2}}\)
    3. \(\displaystyle \lrangle{1,-\dfrac{5}{2}}\)
    4. \(\displaystyle \lrangle{-4,-\dfrac{1}{2}}\)

    Activity 5.1.5.

    This activity will develop the parallelogram rule, a method for visualizing the addition or subtraction of two vectors.
    Consider the vectors \(\textbf{a} = \lrangle{2,0}\) and \(\textbf{b} = \lrangle{1,1}\) for the tasks below.
    (a)
    Find \(\textbf{a} + \textbf{b}\text{.}\)
    (b)
    On a coordinate grid, draw the vectors \(\textbf{a}\) and \(\textbf{b}\) so that their tails are touching.
    (c)
    Create a parallelogram by translating \(\textbf{a}\) and \(\textbf{b}\text{,}\) then include \(\textbf{a} + \textbf{b}\) on the parallelogram. What do you notice about its position on the diagram?
    (d)
    Compute \(\textbf{a} - \textbf{b}\text{.}\) How, if at all, does it fit in the diagram from (c)?

    Vector Properties.

    Let \(\textbf{a}, \textbf{b}\) and \(\textbf{c}\) be three vectors, and let \(k,r\) be scalar values. The following properties hold:
    1. (Commutativity of Vectors) \(\textbf{a} + \textbf{b} = \textbf{b} + \textbf{a}\)
    2. (Associativity of Vectors) \((\textbf{a} + \textbf{b}) + \textbf{c} = \textbf{a} + (\textbf{b} + \textbf{c})\)
    3. (Additive Identity for Vectors) \(\textbf{a} + \textbf{0} = \textbf{a}\)
    4. (Additive Inverse for Vectors) \(\textbf{u} + (-\textbf{u}) = \textbf{0}\)
    5. (Associativity of Scalar Muliplication) \(k(r\textbf{a}) = (kr)\textbf{a}\)
    6. (Distributivity) \((k+r)\textbf{a} = k\textbf{a} + r\textbf{a}\)
    7. (Distributivity) \(k(\textbf{a} + \textbf{b}) = k\textbf{a} + k\textbf{b}\)
    8. (Multiplicative Identity for Vectors) \(1\textbf{a} = \textbf{a}\)
    9. (Multiplicative Zero for Vectors) \(0\textbf{a} = \textbf{0}\)

    Remark 5.1.6.

    A convenient thing about working with vectors is that the same ideas that work in two dimensions work in any number of dimensions. The number of dimensions is equal to the number of components in the vector. For example, the vector \(\lrangle{1,4}\) is two-dimensional because there are only two components, while the vector \(\lrangle{0,1,2,3,4,5}\) is six-dimensional because there are six components.

    Activity 5.1.6.

    For the vectors \(\textbf{a} = \lrangle{-3,5,1}\text{,}\) \(\textbf{b} = \lrangle{1,0,2}\text{,}\) find the following.
    (a)
    \(\textbf{a} + \textbf{b}\)
    1. \(\displaystyle \lrangle{-2,5,3}\)
    2. \(\displaystyle \lrangle{-4,5,-1}\)
    3. \(\displaystyle \lrangle{4,-5,1}\)
    4. \(\displaystyle 6\)
    (b)
    \(-\dfrac{1}{2}\textbf{a} + \dfrac{3}{4}\textbf{b}\)
    (c)
    \(5\textbf{a} \)

    Remark 5.1.7.

    In order to avoid confusion, we talk about a three-dimensional coordinate system using the right-hand rule: when using your right hand, point your index finger out, thumb up, and middle finger perpendicular to both. Your index finger represents the \(y-\)axis, your middle finger represents the \(x-\)axis, and your thumb represents the \(z-\)axis.

    Activity 5.1.7.

    Recall from Activity 5.1.1 that the magnitude or length of a two-dimensional vector \(\textbf{v} = \lrangle{a,b}\) can be found by computing \(\sqrt{a^2 + b^2}\text{.}\)
    (a)
    For a three-dimensional vector \(\textbf{u} = \lrangle{a,b,c}\text{,}\) how might you compute \(||\textbf{u}||\text{?}\)
    (b)
    Find the length of the vector \(\textbf{a} = \lrangle{0,3,4}\text{.}\)
    (c)
    What is the length of the vector \(\dfrac{1}{||\textbf{a}||}\textbf{a}\text{?}\)

    Definition 5.1.8.

    A unit vector is a vector with unit length, i.e. length exactly 1.

    Activity 5.1.8.

    For each vector, find its associated unit vector.
    (a)
    \(\lrangle{7,6}\)
    (b)
    \(\lrangle{-1,0,8}\)
    (c)
    \(\lrangle{0,1}\)
    (d)
    \(\lrangle{1,1,1}\)

    Remark 5.1.9.

    The unit vector \(\textbf{i}\) is the vector whose first component 1 and all others are 0, \(\textbf{j}\) is the one whose second component is 1 with all others 0, and \(\textbf{k}\) is the one whose third component is 1 with all others 0.

    Activity 5.1.9.

    Write each vector as a sum of the standard unit vectors \(\textbf{i}, \textbf{j}, \textbf{k}\text{.}\)
    (a)
    \(\lrangle{-4,3}\)
    1. \(\displaystyle -4\textbf{j} + 3\textbf{k}\)
    2. \(\displaystyle -4\textbf{i} + 3\textbf{k}\)
    3. \(\displaystyle -4\textbf{i} + 3\textbf{j}\)
    4. \(\displaystyle -4\textbf{j} + 3\textbf{i}\)
    (b)
    \(\lrangle{3,1,-5}\)
    1. \(\displaystyle 3\textbf{j} + \textbf{i} - 5\textbf{k}\)
    2. \(\displaystyle 3\textbf{j} + \textbf{k} - 5\textbf{i}\)
    3. \(\displaystyle 3\textbf{i} + \textbf{j} - 5\textbf{k}\)
    4. \(\displaystyle 3\textbf{i} + \textbf{k} - 5\textbf{j}\)
    5. \(\displaystyle 3\textbf{k} + \textbf{i} - 5\textbf{j} \)
    6. \(\displaystyle 3\textbf{k} + \textbf{j} - 5\textbf{i}\)
    (c)
    \(\lrangle{-1,1,0}\)

    Activity 5.1.10.

    The speed of an object is the magnitude of its associated velocity vector. A football is thrown by a player with an initial speed of 82 miles per hour and an angle of elevation of \(25\dc\text{.}\) Determine the velocity vector in miles per hour and express in component form and in terms of the standard unit vectors. Assume the ball is thrown in the \(xz-\)plane.
    (a)
    Which of the following best expresses the horizontal component of the vector?
    1. \(\displaystyle 82\sin(25\dc)\)
    2. \(\displaystyle 25\cos(82\dc)\)
    3. \(\displaystyle 82\cos(25\dc)\)
    4. \(\displaystyle 25\sin(82\dc)\)
    5. \(\displaystyle 82\sin(65\dc)\)
    6. \(\displaystyle 82\cos(65\dc)\)
    (b)
    Which option best describes the vertical component of the vector?
    1. \(\displaystyle 82\sin(25\dc)\)
    2. \(\displaystyle 25\cos(82\dc)\)
    3. \(\displaystyle 82\cos(25\dc)\)
    4. \(\displaystyle 25\sin(82\dc)\)
    5. \(\displaystyle 82\sin(65\dc)\)
    6. \(\displaystyle 82\cos(65\dc)\)
    (c)
    Use your answers to (a) and (b) to write the velocity vector in component form.
    (d)
    Use your answers to (a) and (b) to write the velocity vector using the standard unit vectors. Remember that the ball is thrown in the \(xz-\)plane.
    (e)
    Why would we consider the ball to be thrown in the \(xz-\)plane rather than the \(xy-\)plane or the \(yz-\)plane?
